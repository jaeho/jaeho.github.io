# 하하 어린이 신문 발행 시스템 사용 가이드

`make_news.py` 스크립트를 사용하여 어린이 신문을 발행할 때 사용할 수 있는 주요 파라메터들을 정리한 문서입니다.

## 기본 실행 방법
```bash
python make_news.py
```
별도의 옵션 없이 실행하면 오늘 날짜의 요일에 맞춰 미리 정의된 주제로 신문을 발행합니다.

## 주요 파라메터 (옵션)

| 옵션 | 설명 | 사용 예시 |
| :--- | :--- | :--- |
| `--activity` | **활동 타입 강제 지정**<br>기사 내용과 상관없이 특정 활동을 생성하도록 강제합니다.<br>(`ox_quiz`, `hidden_objects`, `initial_quiz`, `emotion_guess`, `basic`, `coloring`) | `python make_news.py --activity coloring` |
| `--new` | **새로 발행**<br>기존에 생성된 데이터(`data.json`)가 있더라도 이를 무시하고 모든 내용을 새로 생성합니다. | `python make_news.py --new` |
| `--topic` | **주제 직접 입력**<br>요일별 기본 주제가 아닌, 사용자가 원하는 주제로 기사를 작성합니다. | `python make_news.py --topic "우주 여행"` |

## 상세 설명

### 1. `--activity` (활동 타입)
- `ox_quiz`: 기사 내용 기반 OX 퀴즈
- `hidden_objects`: 숨은 그림 찾기
- `initial_quiz`: 초성 퀴즈
- `emotion_guess`: 감정 유추 퀴즈
- `coloring`: 색칠 놀이
- `basic`: 자유 활동 (그림 그리기 등)

### 2. `--new` (새로 발행)
이 옵션을 사용하면 오늘 날짜 폴더(`YYYY-MM-DD`)를 초기화하고 기사 전문과 이미지를 다시 생성합니다. 이미 발행된 신문의 내용이 마음에 들지 않을 때 유용합니다.

### 3. `--topic` (주제 지정)
특정한 뉴스나 교육용 주제를 다루고 싶을 때 사용합니다. 예: `--topic "분리수거 잘하는 법"`. 이 옵션을 사용하면 `--new` 옵션과 관계없이 항상 새로운 기사를 생성합니다.

---
📍 **참고**: 모든 신문 발행이 완료되면 마지막 단계에서 1페이지(`page1.html`)가 자동으로 루트 디렉토리의 `index.html`로 복사되어 메인 페이지로 설정됩니다.
